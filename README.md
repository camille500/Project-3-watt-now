## <a name="started"></a>Getting started

Clone the repo and install the dependencies with
`$ npm install`

To start the app run
`$ npm run start`

Create your own .env file with these variables:

```
MONGODB_URI=<consumer_key>
SESSIONSECRET=<consumer_secret>
```
